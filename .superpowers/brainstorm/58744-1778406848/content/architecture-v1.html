核心设计原则

  • 轻量 Agent 循环 — 自己写核心 while-loop，不用 LangChain Agent 层
  • MCP 作为工具协议标准 — 所有监控数据源统一通过 MCP 适配
  • 企微只做对话入口 — 不参与业务逻辑，纯消息转发
  • 单一 Agent，多 Tool — 一个 Agent，三个 MCP 工具（Prometheus、观测云、FlashDuty）
